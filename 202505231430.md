Surjective mapping

A mapping $\Phi:\mathcal{V}\to \mathcal{W}$ is called **surjective** if $$\Phi(\mathcal V) = \mathcal W$$
- If $\Phi$ is surjective, then every element in $\mathcal{W}$ can be *reached* from $\mathcal{V}$ using $\Phi$
